# America 250 — State & Territory Commissions

Open directory of USA Semiquincentennial (America 250) commissions for all 50 states and the District of Columbia.

## Data

**[`state-commissions.csv`](state-commissions.csv)** — 51 commissions with:

| Field | Description |
|-------|-------------|
| `state_code` | Two-letter state/territory code |
| `state_or_territory` | Full name |
| `commission_name` | Official commission name |
| `commission_url` | Commission website URL |
| `emails` | Contact email(s) |
| `phones` | Contact phone(s) |
| `facebook_url` | Facebook page |
| `instagram_url` | Instagram profile |
| `x_twitter_url` | X/Twitter profile |
| `linkedin_url` | LinkedIn page |
| `youtube_url` | YouTube channel |

Plus additional metadata: page titles, meta descriptions, fetch status, and related links.

## Source

Scraped from [america250.org/our-partners/state-and-territory-commissions](https://america250.org/our-partners/state-and-territory-commissions/) on 2026-03-07, with enrichment via each commission's own website.
